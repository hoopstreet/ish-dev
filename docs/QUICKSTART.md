# 🏀 Hoopstreet Quick Start Guide

## 5-Minute Setup

### 1. Install (One Command)
```bash
curl -fsSL https://raw.githubusercontent.com/hoopstreet/ish-dev/main/setup.sh | sh
```

2. Launch

```bash
/root/menu
```

3. Test Code Executor

Type 1, then paste:

```
# Phase 1
echo "Hello World"

# Phase 2
echo "Current time: $(date)"

# Phase 3
echo "Done!"
END
```

4. Add Your First Credential

Type 6, then 1, enter:

· Name: GITHUB_TOKEN
· Value: your_token_here

5. Connect a GitHub Project

Type 5, then 1, enter:

· URL: https://github.com/yourusername/yourproject
